== Installation ==

** [Detailed User Guide]**

1. Go to plugins -> Add new.
2. Search for "Preciso Bid-Smart ”
3. Click on "Download”
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Manage your Preciso plugin from Left Side Admin Dashboard
